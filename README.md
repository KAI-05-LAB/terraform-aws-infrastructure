# Terraform AWS Infrastructure

## Objective

This project demonstrates Infrastructure as Code (IaC) using Terraform to provision basic AWS infrastructure components.

## AWS Resources

- Virtual Private Cloud (VPC)
- Public Subnet
- Security Group

## Project Files

- main.tf
- variables.tf
- outputs.tf

## Terraform Commands

Initialize Terraform

```bash
terraform init
```

Validate Configuration

```bash
terraform validate
```

Preview Infrastructure

```bash
terraform plan
```

Deploy Infrastructure

```bash
terraform apply
```

Destroy Infrastructure

```bash
terraform destroy
```
